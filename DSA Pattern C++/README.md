## üß† Pattern Explanations

### üîπ Two Pointer Pattern

**Idea:** Use two indices pointing to different parts of a structure to minimize search time.  
**Use Case:** Sorting, searching, linked list manipulation.  
**Common Problems:**

- Pair Sum in Sorted Array
- Remove Duplicates from Sorted Array
- Container With Most Water

---

### üîπ Sliding Window Pattern

**Idea:** Create a ‚Äúwindow‚Äù over data and slide it while updating results dynamically.  
**Use Case:** Subarrays or substrings problems.  
**Common Problems:**

- Maximum Sum Subarray of Size K
- Longest Substring Without Repeating Characters
- Minimum Window Substring

---

### üîπ Fast & Slow Pointer Pattern

**Idea:** Move two pointers at different speeds to detect loops or find midpoints.  
**Common Problems:**

- Linked List Cycle Detection
- Middle of Linked List
- Happy Number

---

### üîπ Binary Search Pattern

**Idea:** Divide and conquer approach on sorted data.  
**Common Problems:**

- Search in Rotated Sorted Array
- Find First and Last Position of Element
- Peak Element in Mountain Array

---

### üîπ Merge Intervals Pattern

**Idea:** Sort intervals and merge overlapping ones for range-based problems.  
**Common Problems:**

- Merge Intervals
- Insert Interval
- Meeting Rooms

---

### üîπ Cyclic Sort Pattern

**Idea:** For problems involving numbers in a range, swap elements to correct positions.  
**Common Problems:**

- Find Missing Number
- Find All Duplicates in an Array
- First Missing Positive

---

### üîπ Top K Elements Pattern

**Idea:** Use heaps to find the most/least frequent or top elements.  
**Common Problems:**

- Kth Largest Element in an Array
- Top K Frequent Words
- K Closest Points to Origin

---

### üîπ Dynamic Programming Pattern

**Idea:** Solve complex problems by storing solutions to subproblems.  
**Common Problems:**

- Fibonacci Sequence
- Longest Increasing Subsequence
- Knapsack Problem
